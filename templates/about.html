ℹ️ About This Project

This project predicts whether a near-Earth object (NEO) is potentially hazardous using a machine learning model.

It uses features such as:

  • Absolute Magnitude
  • Minimum & Maximum Estimated Diameter
  • Relative Velocity
  • Miss Distance

Visual analytics are derived from historical NASA datasets, and real-time asteroid data is pulled from NASA's public API.
